# AI Workfiles

This repository includes the AI-assisted planning and design artifacts used during implementation.

## Included Files

- `requirements.md`: original assessment prompt
- `plan.md`: implementation plan and requirement mapping
- `plan.html`: visualized planning artifact
- `DESIGN.md`: final architecture and tradeoff explanation
- `CLAUDE.md`: concise technical reference for the current implementation
- `AGENTS.md`: copy of `CLAUDE.md` for tool compatibility

## Notes

- These artifacts are documentation-only and are not required at runtime.
- Planning artifacts (`plan.md`, `plan.html`) capture implementation planning stages; final behavior is documented in `README.md` and `DESIGN.md`.
- The production code paths are in `main.py` and `app/`.
